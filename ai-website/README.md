# Baix Landing Page

Static, responsive landing page for **Baix** (road bikes & service).

## Structure
- `index.html` – markup
- `styles.css` – styles
- `main.js` – behavior
- `assets/` – images (hero & bike shots)

## Run locally
Open `index.html` directly, or start a tiny server:

### Python 3
python -m http.server 8000

Then visit http://localhost:8000
